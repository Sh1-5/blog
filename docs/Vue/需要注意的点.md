## data 属性

vue2 中根组件的 data 可以是一个对象或函数，vue3 中根组件的 data 必须是一个函数

## this 指向问题

vue 为所有函数选项绑定了 this，this 指向组件实例，但如果这个函数是箭头函数，那么会指向 window

[this 指向问题](https://mp.weixin.qq.com/s/hYm0JgBI25grNG_2sCRlTA)
