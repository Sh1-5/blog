## provide & inject

用于子孙组件通信

- 父组件不需要知道哪些子组件使用它 provide 的 property

- 子组件不需要知道 inject 的 property 来自哪里

注意：

1.provide 里面拿到的 this 是 undefined，如果想要使用 this，可以将 provide 写成函数，返回一个对象

2.provide 里面的数据默认不是响应式的，如果要变成响应式的，可以使用 computed 包裹

## mitt 全局事件总线

主要用于非子孙组件通信

vue3 从实例中移除了$on，$off 和$once，mitt 是官方推荐的第三方库

安装

```bash
npm install mitt
```

封装一个工具 event-bus.js

```javascript
import mitt from 'mitt'

const emitter = mitt()

export default emitter
```

emitter.emit()发射事件，emitter.on()监听事件，emitter.off()取消监听事件

[具体使用方法](https://github.com/developit/mitt)
