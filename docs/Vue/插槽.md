### 插槽是什么

抽取共性，预留不同

将共同的内容进行封装

将不同的内容使用`slot`元素占位，让外部决定显示什么内容

### 基本使用

```vue
<!-- MySlotCpn组件 -->
<template>
  <div class="my-slot-cpn">
    <h1>组件开始</h1>
    <slot>默认内容</slot>
    <h1>组件结束</h1>
  </div>
</template>

<!-- App组件 -->
<template>
  <div class="app">
    <MySlotCpn>
      <i>Hello Vue</i>
      <div>Hello Vue</div>
    </MySlotCpn>
    <MySlotCpn>
      <strong>Hello Vue</strong>
      <span>Hello Vue</span>
    </MySlotCpn>
    <MySlotCpn>Hello Vue</MySlotCpn>
    <MySlotCpn />
  </div>
</template>
```

`MySlotCpn`组件里的内容会替换整个`slot`元素（若使用组件时没有传入内容，会使用默认内容）

### 多个插槽

每个插槽都会分发

### 具名插槽

```vue
<!-- NavBar组件 -->
<template>
  <div class="nav-bar">
    <div class="left">
      <slot name="left"></slot>
    </div>
    <div class="center">
      <slot name="center"></slot>
    </div>
    <div class="right">
      <slot name="right"></slot>
    </div>
  </div>
</template>

<!-- App组件 -->
<template>
  <div class="app">
    <NavBar>
      <template v-slot:left>
        <button>button</button>
      </template>
      <template v-slot:center>
        <span>span</span>
      </template>
      <template v-slot:right>
        <i>i</i>
      </template>
    </NavBar>
  </div>
</template>
```

会按照`v-slot:`后面的名字插入到对应的位置

一个不带 `name` 的 `slot`，会带有隐含的名字 `default`

### 动态插槽名

当 `slot` 的 `name` 需要动态决定时，使用`v-slot:[dynamicSlotName]`

```vue
<!-- 组件里的name是通过props动态绑定的 -->
<slot :name="name"></slot>

<!-- 使用时通过[]解构 -->
<template v-slot:[name]></template>
```

### 简写

`v-slot:`可以使用`#`替换

### 渲染作用域

父级模版里的所有内容在父级作用域中编译

子模版里的所有内容在子作用域中编译

```vue
<!-- ChildCpn组件 -->
<template>
  <div class="child-cpn">
    <slot></slot>
  </div>
</template>

<script>
export default {
  data() {
    return {
      title: 'Hello Vue'
    }
  }
}
</script>

<!-- App组件 -->
<template>
  <div class="app">
    <ChildCpn>
      <span>{{ title }}</span>
    </ChildCpn>
  </div>
</template>
```

插槽`span`不能访问`title`

### 作用域插槽

要在父级作用域里使用子组件里的数据

```vue
<!-- ShowMessage组件 -->
<template v-for="item in names">
  <slot :item="item"></slot>
</template>

<!-- App组件 -->
<!-- 因为slot没有name，所以name为default，因此可以省略:default -->
<template>
  <ShowNames>
    <template v-slot="slotProps">
      <strong>{{ slotProps.item }}</strong>
    </template>
  </ShowNames>
</template>

<!-- App组件 -->
<!-- 如果是一个默认插槽，可以采用独占默认插槽缩写，省略template，将v-slot挪到组件上 -->
<template>
  <ShowNames v-slot="slotProps">
    <strong>{{ slotProps.item }}</strong>
  </ShowNames>
</template>
```
