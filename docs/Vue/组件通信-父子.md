## 父组件传递给子组件：props

传递数据

1.普通 props

props 的数组写法、对象写法

尽量使用对象写法，限制给全（类型，是否必传/默认值）

有哪些类型？Boolean、Number、String、Array、Object、Date、Function、Symbol

如果类型是数组或对象，那么 default 必须是一个函数，返回一个数组或对象

注意：props 在父组件的大小写问题，推荐在 template 中使用费短横线分隔命名方式（因为 html 对大小写不敏感，虽然.vue 文件不是直接给浏览器处理，但是养成习惯）

2.非 props 的 Attribute，包括 class、style、id 等

- 如果组件有单个根节点，那么会自动继承到根节点，可以在子组件设置 inheritAttrs: false

  常见情况：需要将 attribute 给应用于根元素之外的其他元素，通$attrs 来访问所有非 props 的 attribute

- 组件有多个根节点

  手动绑定到节点上

## 子组件传递给父组件：$emit

触发事件，携带数据

需要定义 emits，子组件通过$emit 发射事件，在父组件通过@监听

发射事件的时候可以携带参数
