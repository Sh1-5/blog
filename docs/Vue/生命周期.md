### 什么是生命周期

每个组件都可能会经历创建、挂载、更新、卸载等一系列过程

### 生命周期函数

组件处于某个过程的回调函数

![组件生命周期](https://staging-cn.vuejs.org/assets/lifecycle.16e4c08e.png)

另外缓存组件还有`activated`和`deactivated`，缓存组件不会卸载

子组件生命周期在父组件`beforeMount`之后开始

### Options API & Composition API

| Options API   | Composition API |
| ------------- | --------------- |
| beforeCreate  | Not needed\*    |
| created       | Not needed\*    |
| beforeMount   | onBeforeMount   |
| mounted       | onMounted       |
| BeforeUpdate  | onBeforeUpdate  |
| updated       | onUpdated       |
| beforeUnmount | onBeforeUnmount |
| unmounted     | onUnmounted     |
| activated     | onActivated     |
| deactivated   | onDeactivated   |
