## 什么是盒子模型

内容，padding、border、margin

<img :src="$withBase('/assets/img/box-model.gif')" alt="box-model">

## 内容

- width/height：宽度/高度， 默认值是 auto

- min-width/min-height：最小宽度/高度

- max-width/max-height：最大宽度/高度

```css
width: 100px;
width: 100%;
width: auto;

min-width: 200px;

max-width: 1000px;
```

min-width、max-width 一般用于移动端适配

## padding

1 个值：省略右、下、左，使用上

2 个值：省略下，使用上，省略左，使用右

3 个值：省略左，使用右

4 个值：顺时针方向，上右下左

```css
padding: 10px;

padding: 10px 20px;

padding: 10px 20px 30px;

padding: 10px 20px 30px 40px;
```

## border

- border-width；宽度
- border-style：样式，solid（实线）、dashed（虚线）
- border-color：颜色

```css
border-width: 1px;

border-style: solid;

border-color: red;

border: 1px solid red;
```

## margin

1 个值：省略右、下、左，使用上

2 个值：省略下，使用上，省略左，使用右

3 个值：省略左，使用右

4 个值：顺时针方向，上右下左

```css
margin: 10px;

margin: 10px 20px;

margin: 10px 20px 30px;

margin: 10px 20px 30px 40px;
```

## box-sizing

- content-box：标准盒子模型，width/height 不包括 padding、border，默认
- border-box：ie 盒子模型，width/height 不包括 padding、border

```css
box-sizing: content-box;

box-sizing: border-box;
```
