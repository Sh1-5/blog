## 什么是选择器？

按照一定的规则选出符合条件的元素

## 通用选择器

```css
* {
  background-color: red;
}
```

匹配所有元素

性能很差，不建议使用

## 元素选择器

```css
div {
  color: red;
}
```

## 类选择器

```css
.container {
  color: red;
}
```

## id 选择器

```css
#header {
  color: red;
}
```

id 唯一，不要重复

不要用标签名作为 id 值

## 属性选择器

```html
<div title="div">我是div元素</div>
```

```css
/* 拥有title属性 */
[title] {
  color: red;
}

/* title属性等于div */
[title='div'] {
  color: red;
}

/* title属性包含div */
[title*='div'] {
  color: red;
}

/* title属性以div开头 */
[title^='div'] {
  color: red;
}

/* title属性以div结尾 */
[title$='div'] {
  color: red;
}

/* title属性包含div，如有其它属性，必须以空格分隔 */
[title~='div'] {
  color: red;
}
```

## 后代选择器

```css
/* .container选择器下的所有span元素 */
.container span {
  color: red;
}
```

## 直接子代选择器

```css
.container > span {
  color: red;
}
```

## 兄弟选择器

```html
<div class="box"></div>
<div></div>
<div></div>
```

```css
/* 相邻兄弟选择器 */
.box + div {
  color: red; /* 第二个div */
}

/* 普遍兄弟选择器 */
.box ~ div {
  color: red; /* 第二、三个div */
}
```

## 交集选择器

```css
/* 既是div元素，也有container类 */
div.container {
  color: red;
}
```

## 并集选择器

```css
body,
h1 {
  margin: 0;
}
```

## 伪类

元素的状态

```css
div:hover {
  color: red;
}
```

动态伪类：:link（a 元素，未访问的链接）、:visited（a 元素，已访问的链接）、:hover（常用，必须放在:link 和:visited 后面）、:active（必须放在:hover 后面）、:focus

目标伪类：:target（锚点）

语言伪类：:lang()

元素状态伪类：:enabled、:disabled、:checked

结构伪类：:nth-child()、:nth-last-child()、:nth-type()、:nth-last-of-type()、:first-child()、:last-child()、:first-of-type()、:last-of-type()、:root、:only-child()、:only-of-type()、:empty()

否定伪类：:not()
