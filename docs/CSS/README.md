## 认识 CSS

Cascading Style Sheet，层叠样式表，是为网页添加样式的代码

CSS 不是真正的编程语言，是一门样式表语言

## CSS 的历史

1996，CSS1

1998，W3C，CSS2

2011，W3C，CSS3（分模块发布）

CSS 的出现是为了美化 CSS，并且让结构（HTML）与样式（CSS）分离

## 编写方法

- 内联样式（放在全局属性 style 里面）
- 内部样式（放在 head 元素下面的 style 元素里面）
- 外部样式（通过 link 元素从外部引入 CSS 文件）
  `<link rel="stylesheet" href="" />`
